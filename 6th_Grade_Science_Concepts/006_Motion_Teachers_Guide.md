# Motion Teacher's Guide

## Answer Key

1. Motion is a change in position over time.

2. A reference point is something used to tell whether an object has moved.

3. Motion must be described compared with something else because an object may be moving compared with one reference point but not moving compared with another. For example, a backpack on a train is still compared with the passenger but moving compared with a person beside the track.

4. Position tells where an object is.

5. Direction is important because two objects can move the same distance but end up in different places if they move in different directions.

6. Distance tells how much ground an object has covered. Displacement tells how far and in what direction an object is from its starting point.

7. The distance is 20 meters. The displacement is 0 meters because the person ends where he started.

8. Speed tells how fast an object moves.

9. The basic formula for speed is speed = distance / time.

10. Velocity is speed in a particular direction.

11. Speed tells how fast something moves. Velocity tells both speed and direction.

12. In science, acceleration means any change in velocity.

13. An object can accelerate by speeding up, slowing down, or changing direction.

14. An unbalanced force changes an object's motion.

15. Inertia is an object's resistance to changes in motion.

16. An object with more mass has more inertia, so it is harder to start moving, stop, or change direction.

17. Answers may vary. Good examples include straight-line motion, curved motion, circular motion, vibrating motion, rolling motion, or spinning motion.

18. Relative motion means that motion can look different depending on the observer or reference point.

19. A distance-time graph can show how distance changes over time. It can help show whether an object is stopped, moving steadily, speeding up, or slowing down.

20. Answers may vary. Good examples include walking, running, throwing a ball, riding a bicycle, driving a car, opening a door, water flowing in a river, leaves falling, planets orbiting, or blood moving through the body.

## Activity Suggestions

1. **Reference Point Walk**  
   Have one student walk across the room while classmates describe the motion using different reference points, such as the door, a desk, or another student.

2. **Distance vs. Displacement Path**  
   Mark a simple path on the floor with tape. Students walk out and back, then compare total distance traveled with displacement from the starting point.

3. **Speed Calculation Trial**  
   Time students walking or rolling a toy car across a measured distance. Have them calculate speed using speed = distance / time.

4. **Velocity Direction Cards**  
   Give students cards such as "3 meters per second north" or "5 meters per second east." Ask them to identify speed and direction, then explain why each card describes velocity.

5. **Acceleration Sort**  
   Provide examples such as a car speeding up, a bicycle braking, a runner turning a corner, and a book resting on a table. Students sort which examples show acceleration and explain why.

6. **Inertia Demonstration**  
   Place an index card over a cup with a coin on top. Flick the card quickly so the coin drops into the cup. Use the demonstration to discuss inertia and changes in motion.

7. **Motion Graph Stories**  
   Show simple distance-time graphs and ask students to write a short story for each one, such as "the walker stood still, then moved steadily, then stopped."

8. **Types of Motion Stations**  
   Set up small stations showing straight-line motion, circular motion, rolling motion, and vibrating motion. Students identify each type and describe the forces that may be involved.
